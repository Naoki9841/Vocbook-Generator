# Vocbook-Generator

chromeのbookmark機能を利用して、google spreadsheetに英単語帳を作成できます。

手順は
1. weblioで単語を調べたら、そのurlをbookmarkに保存
2. ある程度貯まったら、`make_vocbook.py`をPython3で実行する
